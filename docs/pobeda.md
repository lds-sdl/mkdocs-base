# Catalogue de 1953

- 52-K : Diamètre du boîtier 26,9 mm. marqueurs et chiffres noirs. Aiguilles blues
- 103-K : Diamètre du boitier 32mm, chiffres de l'échelle des heures dorés bordés de noir. aiguilles dorées.
